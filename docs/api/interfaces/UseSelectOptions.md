# Interface: UseSelectOptions

Configuration options for the [useSelect](../functions/useSelect.md) hook.

## Properties

### isDisabled

> **isDisabled**: `boolean`

If true, the hook will not respond to any keyboard input.

---

### state

> **state**: [`SelectState`](SelectState.md)

The current state of the select component, as returned by [useSelectState](../functions/useSelectState.md).
This hook uses the state's navigation and selection functions.
